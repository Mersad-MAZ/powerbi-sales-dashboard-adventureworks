# Power BI Sales Dashboard | AdventureWorks

## Overview

This project showcases an interactive Power BI dashboard built using the AdventureWorks dataset. The dashboard provides insights into sales performance, customer behavior, product trends, and business KPIs through interactive visualizations and data modeling techniques.

The project demonstrates data preparation, modeling, DAX calculations, and dashboard design using Power BI.

---

## Project Objectives

* Monitor sales performance across different time periods.
* Analyze customer purchasing behavior.
* Evaluate product and category performance.
* Track key business KPIs.
* Provide interactive business insights through visual reporting.

---

## Dataset

The dashboard is built using the AdventureWorks dataset and includes:

* Sales Data
* Customer Data
* Product Data
* Product Categories
* Product Subcategories
* Territory Information
* Returns Data
* Calendar Table
